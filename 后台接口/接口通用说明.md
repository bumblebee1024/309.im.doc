# 接口通用说明

## 网络协议
* 使用HTTP的POST和Get方式
* 返回数据为Json格式，通常情况下包含以下两个字段
<pre><code>
{
	"flag":0,
	"message":"success"
}
</code></pre>

flag: 标识状态码。

message： 状态码对应携带的信息。
